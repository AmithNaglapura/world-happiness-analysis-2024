# World Happiness Report 2024 Analysis

## Project Overview
This project explores the factors influencing global happiness scores using the 2024 World Happiness Report dataset. The analysis includes data cleaning, exploratory data analysis (EDA), and regression modeling.

## Key Features
- **Exploratory Data Analysis:** Visualizing correlations between GDP, Social Support, and Happiness.
- **Predictive Modeling:** Using Linear Regression to understand the impact of various life factors.
- **Visualizations:** Heatmaps and scatter plots generated using Seaborn and Matplotlib.

## Dataset
The data is sourced from the World Happiness Report 2024 (`WHR2024.csv`), covering 143 countries.
